# zsh-yeoman

[zsh](https://www.zsh.org/) plugin for interacting w/ [yeoman](https://yeoman.io/)

## Installation


<details>
  <summary>zimfw</summary>

  ```sh
  zmodule $ZMODULES_DIR/zsh-yeoman/ # --if '[[ \$+commands[yo] ]]'
  ```
</details>


### Verifying Installation

```sh
Yv
# -> 6.0.0
```
